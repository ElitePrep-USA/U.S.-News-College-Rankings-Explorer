# U.S. News Historical Rankings Explorer

## GitHub Pages deployment
Upload the contents of this folder to the root of a GitHub repository:

- `index.html`
- `data/rankings.json`
- `data/rankings_update_template.csv`
- `docs/ADDING_NEW_RANKINGS.md`

Then enable GitHub Pages from the repository's main branch and root folder.

## Important
Do not open `index.html` only as a local file and expect the JSON to load. Use GitHub Pages or a local web server.
